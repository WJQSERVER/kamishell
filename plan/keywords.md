# Kamishell 关键字与操作符表

## 1. 编程控制关键字 (类 Go)
用于脚本逻辑控制和结构定义。

| 关键字 | 描述 |
| :--- | :--- |
| `func` | 定义函数 |
| `return` | 从函数返回 |
| `if` | 条件判断 |
| `else` | 条件分支 |
| `for` | 循环结构 |
| `range` | 迭代集合 |
| `go` | 启动 Goroutine (后台执行) |
| `var` | 变量声明 |
| `const` | 常量定义 |
| `import` | 导入模块或包 |
| `switch` | 分支选择 |
| `case` | 开关分支 |
| `default` | 默认分支 |
| `defer` | 延迟执行 |
| `exec` | 执行字符串形式的命令 (避开关键字冲突) |

## 2. Shell 专用关键字
用于管理 Shell 环境和兼容性。

| 关键字 | 描述 |
| :--- | :--- |
| `export` | 设置或导出环境变量 |
| `alias` | 定义命令别名 |
| `source` | 在当前环境中执行脚本 |
| `exit` | 退出当前 Shell |

## 3. 字面量
| 关键字 | 描述 |
| :--- | :--- |
| `true` | 布尔真 |
| `false` | 布尔假 |
| `nil` | 空值/错误空值 |

## 4. 特殊操作符与符号
| 符号 | 描述 |
| :--- | :--- |
| `:=` | 短变量声明与赋值 |
| `=` | 赋值 |
| `|` | 管道操作符 |
| `>` | 输出重定向 (覆盖) |
| `>>` | 输出重定向 (追加) |
| `&` | 传统 Shell 后台运行符 |
| `$` | 变量插值符号 |
| `&&` | 逻辑与 / 命令成功后执行 |
| `||` | 逻辑或 / 命令失败后执行 |
| `!` | 逻辑非 |
| `!=` | 不等于 |
| `==` | 等于 |
| `+`, `-`, `*`, `/` | 算术运算符 |

## 5. 内置核心命令
这些命令由 Kamishell 内部实现，以确保跨平台一致性。

| 命令 | 描述 |
| :--- | :--- |
| `print` | 打印到标准输出 (替代 `echo`) |
| `ls` | 列出目录内容 (支持 `-a`, `-l`, `-h`, `-F`) |
| `cd` | 切换工作目录 (待实现) |
| `pwd` | 显示当前工作目录 (待实现) |
| `type` | 显示命令类型 |
| `help` | 显示帮助信息 |
