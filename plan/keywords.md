# Kamishell 关键字与操作符表

## 1. 编程控制关键字 (类 Go)
用于脚本逻辑控制和结构定义。

| 关键字 | 描述 |
| :--- | :--- |
| `func` | 定义函数 |
| `return` | 从函数返回 |
| `if` | 条件判断 |
| `else` | 条件分支 |
| `for` | 循环结构 |
| `range` | 迭代集合 |
| `go` | 启动 Goroutine (后台执行) |
| `var` | 变量声明 |
| `const` | 常量定义 |
| `import` | 导入模块或包 |
| `switch` | 分支选择 |
| `case` | 开关分支 |
| `default` | 默认分支 |
| `defer` | 延迟执行 |

## 2. Shell 专用关键字
用于管理 Shell 环境和兼容性。

| 关键字 | 描述 |
| :--- | :--- |
| `export` | 设置或导出环境变量 |
| `alias` | 定义命令别名 |
| `source` | 在当前环境中执行脚本 |
| `exit` | 退出当前 Shell |

## 3. 字面量
| 关键字 | 描述 |
| :--- | :--- |
| `true` | 布尔真 |
| `false` | 布尔假 |
| `nil` | 空值/错误空值 |

## 4. 特殊操作符与符号
| 符号 | 描述 |
| :--- | :--- |
| `:=` | 短变量声明与赋值 |
| `=` | 赋值 |
| `|` | 管道操作符 |
| `>` | 输出重定向 (覆盖) |
| `>>` | 输出重定向 (追加) |
| `&` | 传统 Shell 后台运行符 |
| `$` | 变量插值符号 |
| `&&` | 逻辑与 / 命令成功后执行 |
| `||` | 逻辑或 / 命令失败后执行 |
| `!` | 逻辑非 |
| `!=` | 不等于 |
| `==` | 等于 |
| `+`, `-`, `*`, `/` | 算术运算符 |

## 5. 内置核心命令 (预留)
虽然技术上是命令，但在解析时具有高优先级。
`cd`, `pwd`, `print`, `type`, `help`
